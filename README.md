# MovieAwardsGraphDB: Пример графовой базы данных для фильмов и наград

## Описание проекта

Репозиторий содержит учебный пример построения и использования графовой базы данных в Microsoft SQL Server на примере наград и участников кинофестиваля. Проект включает создание и заполнение графовой базы, примеры запросов на языке T-SQL с использованием операторов `MATCH` и `SHORTEST_PATH`, схему данных в draw.io, визуализацию в Power BI, а также собственную задачу на графовую БД.

## Содержимое репозитория

- **/MovieAwardsGraphDB.bak** — резервная копия итоговой базы данных.
- **/MovieAwardsGraphDB.sql** — скрипт создания и наполнения графовой базы данных, а также примеры запросов.
- **/diagrams.drawio** — файл схемы данных, каждая диаграмма на отдельной странице.
- **/MovieAwardsGraphDB.pbix** — файл Power BI с визуализацией графа с помощью Force-Directed Graph.
- **/images/** — папка с изображениями для Force-Directed Graph (Power BI).
- **/task.txt** — описание вашей собственной задачи для графовой базы данных.
- **/problem_statement.txt** — условие учебной задачи (см. ниже).
- **README.md** — этот файл.

## Условие задачи

1. Создать не менее 3 таблиц-узлов (например, фильмы, люди, награды).
2. Создать не менее 3 таблиц-ребер (например, участвовал, номинирован, выиграл).
3. Заполнить таблицы узлов минимум 10 строками каждая.
4. Заполнить таблицы рёбер связями между узлами.
5. Написать не менее 5 различных запросов с использованием MATCH.
6. Написать не менее 2 различных запросов с использованием SHORTEST_PATH (с шаблонами "+" и "{1,n}").
7. Нарисовать диаграммы в draw.io (каждая диаграмма — отдельная страница).
8. Визуализировать граф в Power BI с помощью Force-Directed Graph.
9. Предложить свою задачу на графовую БД.

## Как использовать

1. **Восстановите базу из файла** `MovieAwardsGraphDB.bak` или выполните скрипт `MovieAwardsGraphDB.sql` в Microsoft SQL Server (2019+ с поддержкой graph tables).
2. **Откройте файл** `diagrams.drawio` на [draw.io](https://www.drawio.com/) — каждая страница содержит отдельную диаграмму (структура узлов, связей, примеры графов).
3. **Откройте файл** `MovieAwardsGraphDB.pbix` с помощью Power BI — для визуализации Force-Directed Graph (в настройках укажите путь к вашей базе данных).
4. **Папка** `/images` содержит изображения для Force-Directed Graph (например, иконки фильмов, людей и пр.).
5. **Ознакомьтесь с файлом** `task.txt` — содержит вашу собственную задачу на графовую базу данных.

## Примеры SQL-запросов

В скрипте `MovieAwardsGraphDB.sql` представлены примеры запросов:

- Люди, номинированные на лучшую женскую роль.
- Люди с номинациями в технических категориях.
- Участники фильма "Анора" и их роли.
- Все режиссёры фильмов, номинированных на "Лучший фильм".
- Актёры из США, участвовавшие в фильмах 2024 года.
- Связи между Шоном Бэйкером и другими участниками через фильмы (SHORTEST_PATH).
- Все награды/номинации для Мэдисон Майки — лично и через фильмы.
